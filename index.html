SMART Visitor Recognition

This project focuses on creating a smart visitor recognition system that provides notifications to the owner through the Internet whenever a person comes at the door. When a person comes at the door, camera sensor captures his/her image and sends it to the ML face recognition module. It recognises the face of the user with any of the allowed members and matches the entry timing. Furthermore, we have also provided a time-based heuristic which allows entry only under desired timings. The advantage gained by choosing this system over similar current systems is that it provides automatic entry in case the person is recognised and is trying to enter within desired timings. In all other cases, the user receives a notification to allow/deny. The developed system provides accurate judgements under different test scenarios.

Intelligent Farm Irrigation System

The goal is to create an intelligent irrigation system that uses servo motors to manage water flow based on a Machine Learning algorithm’s forecast. To anticipate the water flow, this algorithm uses the humidity and temperature data produced by the sensor associated with each individual servo motor sensor. In addition, the water flow is only controlled during the day. During night time, system is switched off.
